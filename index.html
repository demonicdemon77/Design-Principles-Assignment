Definitions

Contrast
Making elements different to highlight different areas of the site.
Repetition
Repeating colors, fonts, and other design elements to create consistency.
Alignment
Using organization to make sure everything in the layout is aligned.
Proximity
Grouping related items together to improve consistency.
